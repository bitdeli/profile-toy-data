
Toy Data
--------

Generates customer activity from an imaginary deli. Great for trying out Bitdeli.

- **Source:** Synthetic data.

- **Historical data:** N/A.

- **Retention:** All data available in all plans.

- **Schema:**
    .. code-block:: python

        {
          "events": [
            {
              "product": "BLT",
              "uid": "00002",
              "country": "MX",
              "price": 9,
              "event": "buy",
              "day": "Wed"
            }]
        }

    Customer actions from the imaginary deli.

- **Update interval:** N/A.
